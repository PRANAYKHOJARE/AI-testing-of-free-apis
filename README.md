# 🚀 AI Testing of Free APIs

This project demonstrates testing and integration of various **free AI APIs** such as OpenRouter, Groq, and others using tools like Postman.

---

## 📌 Features

- 🔑 API Key Testing (OpenRouter, Groq)
- 📡 API Request Handling using Postman
- ⚡ Fast LLM Response Testing
- 🧪 Real-world API experimentation

---

## 🛠️ Tech Stack

- Postman
- REST APIs
- OpenRouter AI
- Groq API
- JSON
